# Knight journey

A platformer game was created using the Unity game engine and the C# programming language. 
<br/>
<p align="center"><img src="https://github.com/user-attachments/assets/0a573094-5229-45be-9375-8b1eca128bc6" alt="Screenshot 1" width="450"/></p>


The game features implemented algorithms for the player control system, platform movement system, collecting items and character death mechanics. 
<br/>
<p align="center"><img src="https://github.com/user-attachments/assets/307a0bcb-fe87-40ec-a949-d757ba20a47e" alt="Screenshot 3" width="450"/></p>

The visuals were developed using both free assets and custom-designed elements. Also there were added background music and sounds after interactions with objects.
<br/>
<p align="center"><img src="https://github.com/user-attachments/assets/fc5a2230-2fd8-41ef-853a-b0beffd615ba" alt="End of the game" width="450"/></p>

During the development process, experience was gained in application development and testing, the fundamentals of object-oriented programming (OOP) were learned, and the full project development cycle was understood.
